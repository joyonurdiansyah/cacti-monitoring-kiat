# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.3.x   | :white_check_mark: |
| 1.2.x   | :white_check_mark: |
| 1.1.x   | :x:                |
| 1.0.x   | :x:                |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Please use either GitHub's private vulnerability reporting or send an email to developers@cacti.net.  We will aim to respond within a few days and accept any PoCs that demonstrate how to test for the vulnerability.

We will remove any PoC from published advisories where possible and will advise to this effect when doing so.
